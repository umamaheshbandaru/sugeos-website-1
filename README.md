
# Sugeos Tech Systems

Official website for Sugeos Tech Systems, a modern software development agency based in Guntur, Andhra Pradesh, India.

## Deployment to GitHub Pages

This site is optimized for GitHub Pages hosting using a **No-Build React Architecture** (ESM Modules).

### How to host:
1.  **Create a Repository**: Create a new repo on GitHub (e.g., `sugeos-website`).
2.  **Upload Files**: Push all the files in this project directly to the `main` branch.
3.  **Activate Pages**:
    -   Navigate to **Settings** > **Pages** in your GitHub repository.
    -   Under **Build and deployment**, select "Deploy from a branch".
    -   Choose `main` and the folder `/ (root)`.
    -   Click **Save**.
4.  **Live Site**: Your site will be live at `https://[username].github.io/[repo-name]/`.

### Technology Stack:
- **Core**: React 19 (Loaded via ESM)
- **Styling**: Tailwind CSS (Runtime CDN)
- **Routing**: React Router 7 (HashRouter for URL stability)
- **Persistence**: LocalStorage for Admin Data
- **Icons**: Lucide React
